# JetCheck 1.3 Demo 示例项目口径说明

本文档用于固化当前 Demo 的示例项目口径，避免在演示和沟通中反复口头说明。

## 总体约定

- 以下 4 个项目为系统默认示例项目基线。
- 示例图统一来自 `sample-images` 目录。
- 说明中的“图像获取 -> 图像处理 -> 图像检测”即为配置链路。
- 客户端演示范围已去除“采图工具/采图记录”独立模块；采样演示统一在检测工具运行链路内完成。

## 项目 1：安全座椅

- 业务目标：检测零件上的卡扣和螺杆。
- 图像获取：
  - 卡扣图像（相机输入）对应 `sample-images/安全座椅-卡扣kakou.png`
  - 螺杆图像（相机输入）对应 `sample-images/安全座椅-螺杆.bmp`
- 图像处理：
  - 卡扣图像：全图处理
  - 螺杆图像：全图处理
- 图像检测：
  - 卡扣图像：卡扣外观分类
  - 螺杆图像：同一张图分别执行左模版尺寸检测与右模版尺寸检测

## 项目 2：马斯特 X 光

- 业务目标：检测零件上的气泡、熔断等缺陷。
- 图像获取：
  - 单张图像（接口输入）对应 `sample-images/马斯特X光.bmp`
- 图像处理：
  - 使用“加热丝零件识别模型”做模型 ROI，识别图中零件 ROI 位置
- 图像检测：
  - 在指定零件类型上进行缺陷检测

## 项目 3：血氧纱布

- 业务目标：检测纱布毛丝、黑点等缺陷。
- 图像获取：
  - 单张图像（相机输入）对应 `sample-images/血氧纱布.jpg`
- 图像处理：
  - 手绘 ROI 并按 16 等分配置
- 图像检测：
  - 选择“纱布缺陷检测”模型进行检测

## 项目 4：瑾辰汽车饰件

- 业务目标：检测卡扣安装有无和焊点焊接有无。
- 图像获取：
  - 左图（相机输入）对应 `sample-images/瑾辰-左.png`
  - 右图（相机输入）对应 `sample-images/瑾辰-右.png`
- 图像处理：
  - 在左图手绘 2 个 ROI（卡扣 ROI、焊点 ROI）
  - 在右图手绘 2 个 ROI（卡扣 ROI、焊点 ROI）
  - 总计 4 个图像处理实例
- 图像检测：
  - 两个卡扣 ROI 使用卡扣模型
  - 两个焊点 ROI 使用焊点模型
