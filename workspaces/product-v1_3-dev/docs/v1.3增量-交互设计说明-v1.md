# JetCheck 新增功能交互设计说明 v1

## 1. 文档定位

本文档用于描述本期新增功能的页面结构、交互流程、关键控件行为、状态反馈、校验规则与异常处理方式，作为原型设计、视觉设计与前端实现的依据。

本文档基于以下需求范围：

1. 客户端账号登录、自动绑定、客户端本地解绑
2. 平台端客户端额度与客户端列表管理
3. 客户端心跳上报与在线状态展示
4. 客户端模型管理
5. 客户端相机管理
6. 客户端存储管理
7. 检测工具整体重构
8. 检测记录改造
9. 采图记录与检测记录导出方式调整

## 2. 设计原则

1. 搭建态与运行态分离。
2. 一线操作优先展示结果，弱化复杂配置。
3. 配置类页面优先展示结构关系，减少一次性长表单。
4. 所有高风险操作均需提供明确提示与确认。
5. 所有阻断、失败、禁用状态均需可见、可解释。

## 3. 全局交互约定

### 3.1 导航结构

平台端与客户端分别独立导航。

客户端本期涉及页面：

1. 登录页
2. 检测工具
3. 相机管理
4. 设置
5. 检测记录
6. 采图记录

### 3.2 操作反馈

统一采用以下反馈方式：

1. 即时成功反馈：轻提示
2. 阻断型失败反馈：表单错误提示或弹窗提示
3. 高风险操作：二次确认弹窗
4. 长耗时操作：按钮 loading、列表项状态变化、必要时展示进度信息

### 3.3 通用状态

列表页、详情页、弹窗统一支持以下状态：

1. 默认态
2. 空态
3. 加载态
4. 错误态
5. 禁用态

### 3.4 文案规则

1. 成功文案直接描述结果，不带解释性句式。
2. 失败文案优先描述失败原因，其次描述可执行动作。
3. 禁用按钮需配合悬浮说明或邻近提示文案。

## 4. 平台端交互设计

### 4.1 客户端管理页

#### 4.1.1 页面目标

展示企业账号下的客户端额度、客户端列表与在线状态。

#### 4.1.2 页面结构

页面由以下区域组成：

1. 顶部摘要区
   - 客户端总额度
   - 已使用数量
   - 剩余数量
2. 筛选区
   - 状态筛选
   - 关键词搜索
3. 列表区
   - 客户端名称
   - 硬件识别码
   - 绑定时间
   - 最近心跳时间
   - 在线状态
   - 最近离线时间

#### 4.1.3 交互规则

1. 状态筛选支持全部、在线、离线。
2. 关键词支持按客户端名称、硬件识别码搜索。
3. 最近心跳信息在 10 分钟内时展示在线，否则展示离线。
4. 本页不提供新增客户端入口。
5. 本页不提供解绑入口。

#### 4.1.4 状态设计

1. 无客户端记录时展示空态。
2. 加载失败时展示重试入口。
3. 无筛选结果时展示空结果态。

## 5. 客户端登录与绑定交互设计

### 5.1 登录页

#### 5.1.1 页面目标

完成账号登录与设备自动绑定校验。

#### 5.1.2 页面结构

1. 账号输入框
2. 密码输入框
3. 登录按钮
4. 错误提示区

#### 5.1.3 交互流程

1. 用户输入账号密码后点击登录。
2. 登录按钮进入 loading 状态。
3. 登录成功后，客户端自动读取硬件识别码并发起设备校验。
4. 已绑定设备直接进入系统。
5. 未绑定设备在额度充足时自动绑定并进入系统。
6. 额度不足时停留当前页并提示。

#### 5.1.4 校验与反馈

1. 账号为空时不可提交。
2. 密码为空时不可提交。
3. 登录失败时提示账号或密码错误。
4. 网络异常时提示无法连接云端服务。
5. 自动绑定成功时提示当前设备已绑定到企业账号。
6. 额度不足时提示当前企业客户端数量已达上限。

### 5.2 客户端解绑

#### 5.2.1 入口位置

客户端设置页中的客户端信息区域。

#### 5.2.2 交互流程

1. 用户点击解绑按钮。
2. 弹出二次确认弹窗。
3. 用户确认后发起解绑。
4. 解绑成功后退出当前可用状态。
5. 页面跳转回登录入口或未登录状态页。

#### 5.2.3 提示规则

1. 解绑成功后提示当前客户端已解绑，需重新登录并重新绑定后方可继续使用。
2. 解绑失败时提示失败原因。

## 6. 模型选择交互设计

### 6.1 入口位置

模型管理不提供独立一级页面。

模型选择入口位于以下场景：

1. 检测工具搭建态中新增或编辑检测实例
2. 检测工具搭建态中修改检测实例使用模型

### 6.2 交互形式

模型选择采用二级抽屉形式承载。

抽屉包含两个视图：

1. 本地模型
2. 云端模型

### 6.3 本地模型视图

#### 6.3.1 页面结构

1. 本地模型列表
   - 模型名称
   - 模型版本编号
   - 来源
   - 添加时间
   - 引用状态
2. 操作区
   - 选择模型
   - 离线导入
   - 删除

#### 6.3.2 交互规则

1. 默认优先进入本地模型视图。
2. 点击模型行可选中当前模型。
3. 已选中模型需有明确选中态。
4. 点击确认后回填到检测实例表单。
5. 离线导入后校验模型版本编号唯一性。
6. 版本编号重复时，拒绝导入并提示。
7. 已被检测工具引用的模型删除按钮为禁用态。
8. 删除前弹出二次确认弹窗。

### 6.4 云端模型视图

#### 6.4.1 页面结构

1. 筛选区
   - 模型名称搜索
   - 场景类型筛选
   - 版本编号搜索
2. 云端模型列表
   - 模型名称
   - 场景类型
   - 最近更新时间
   - 版本列表

#### 6.4.2 版本项交互

每个版本项展示：

1. 版本编号
2. 训练完成时间
3. 下载状态
4. 下载按钮

#### 6.4.3 下载交互

1. 未下载时展示“下载到本地”。
2. 下载中时按钮进入 loading，不可重复点击。
3. 已下载时展示“已下载”并禁用。
4. 下载成功后更新本地模型列表，并支持切回本地模型视图进行选择。
5. 下载失败后展示失败提示并支持重试。

### 6.5 空态与异常

1. 本地模型为空时展示空态和导入入口。
2. 云端模型查询无结果时展示空结果态。
3. 云端加载失败时展示重试入口。

## 7. 相机管理交互设计

### 7.1 相机管理列表页

#### 7.1.1 页面结构

1. 筛选区
   - 相机名称搜索
   - 相机 ID 搜索
   - 厂商筛选
   - 型号筛选
2. 操作区
   - 添加相机
   - 刷新
3. 列表区
   - 相机名称
   - 相机 ID 或设备标识
   - 厂商
   - 型号
   - 状态
   - 操作

#### 7.1.2 列表规则

1. 相机名称为空时，默认显示相机 ID 或设备序列号。
2. 状态显示为离线、空闲、占用。
3. 列表不提供连接、断开按钮。

#### 7.1.3 行操作

1. 编辑名称
2. 预览
3. 参数组管理
4. 删除

#### 7.1.4 删除规则

1. 已被检测工具引用时，删除按钮禁用。
2. 未被引用时，删除前弹出二次确认。

### 7.2 添加相机弹窗

#### 7.2.1 页面结构

1. 搜索或扫描触发按钮
2. 搜索结果列表
   - 选择框
   - 相机名称
   - 相机 ID 或设备标识
   - 厂商
   - 型号
   - IP 地址
3. 确认添加按钮

#### 7.2.2 搜索结果规则

1. 搜索结果中的相机名称为空时，默认显示相机 ID 或设备序列号。
2. 已添加到管理列表的设备不可重复选择。
3. 已添加设备在结果列表中展示为禁用态。
4. 已添加设备需保留可识别标识。

#### 7.2.3 添加交互

1. 支持多选添加。
2. 未选择任何设备时，确认按钮不可提交或点击后提示。
3. 添加成功后关闭弹窗并刷新相机列表。

### 7.3 编辑相机名称

#### 7.3.1 交互形式

弹窗或行内编辑。

#### 7.3.2 交互规则

1. 支持输入自定义名称。
2. 清空名称时，保存后恢复为默认展示规则。

### 7.4 参数组管理

#### 7.4.1 页面结构

参数组管理采用弹窗或抽屉形式，包含：

1. 参数组列表
2. 新增参数组
3. 编辑参数组
4. 删除参数组
5. 预览调试区域

#### 7.4.2 表单规则

1. 参数组表单按当前相机品牌动态展示参数项。
2. 不同品牌相机显示不同参数结构。
3. 参数组名称为必填项。

#### 7.4.3 删除规则

1. 已被检测工具引用的参数组不可删除。
2. 未被引用的参数组删除前需二次确认。

## 8. 存储管理交互设计

### 8.1 入口位置

存储管理作为设置页中的子页面承载。

### 8.2 存储管理页

#### 8.2.1 页面结构

1. 空间概览区
   - 当前磁盘剩余空间
   - 检测记录图片占用
   - 采图记录图片占用
   - 模型文件占用
   - 其它业务数据占用
2. 阈值设置区
   - 提醒阈值
   - 阻断阈值
   - 保存按钮
3. 批量清理区
   - 数据类型选择
   - 清理范围说明
   - 批量清理按钮

### 8.3 阈值交互

1. 阻断阈值不得高于提醒阈值。
2. 阻断阈值低于最小安全下限时不可保存。
3. 保存成功后提示设置已更新。

### 8.4 提醒与阻断

#### 8.4.1 提醒阈值

1. 低于提醒阈值时立即提示。
2. 后续触发核心操作时继续提示。
3. 提醒不阻断当前操作。

#### 8.4.2 阻断阈值

1. 被阻断功能入口展示禁用态。
2. 被阻断功能入口邻近展示阻断原因。
3. 用户点击禁用操作时，可展示说明性提示。
4. 当检测、采图、模型下载、模型导入等相关功能运行中达到阻断阈值时，系统强制中断当前任务。
5. 强制中断后需弹出明确提示，说明中断原因和恢复方式。

### 8.5 批量清理

#### 8.5.1 交互流程

1. 用户选择数据类型。
2. 系统展示待清理范围。
3. 用户点击清理。
4. 弹出二次确认弹窗。
5. 清理完成后刷新空间概览。

#### 8.5.2 清理限制

1. 正在被使用中的关键数据不可纳入可清理范围。
2. 清理完成后需反馈释放空间结果。

## 9. 检测工具交互设计

### 9.1 检测工具列表页

#### 9.1.1 页面结构

1. 工具卡片列表
   - 工具名称
   - 配置状态
   - 编辑入口
   - 新建工具入口

#### 9.1.2 交互规则

1. 点击卡片主体进入运行态。
2. 点击编辑进入搭建态。
3. 新建工具进入搭建向导第一页。

### 9.2 搭建态总体结构

#### 9.2.1 页面结构

搭建态采用三步向导：

1. 步骤一：图像获取
2. 步骤二：图像处理
3. 步骤三：图像检测

#### 9.2.2 通用交互

1. 每步展示当前步骤实例列表。
2. 支持新增、编辑、删除当前步骤实例。
3. 支持上一步、下一步、完成。
4. 未满足基础条件时，下一步按钮不可进入或点击后提示。

### 9.3 步骤一：图像获取

#### 9.3.1 页面结构

1. 图像获取实例列表
2. 新建图像获取实例按钮

#### 9.3.2 实例类型

1. 相机获取
2. 接口获取

#### 9.3.3 图像获取实例弹窗

表单包含：

1. 实例名称
2. 获取类型
3. 相机关联信息或接口信息
4. 参数信息
5. 示例图片上传与预览

### 9.4 步骤二：图像处理

#### 9.4.1 页面结构

1. 图像处理实例列表
2. 图像获取实例辅助查看区
3. 新增图像处理实例按钮

#### 9.4.2 处理方式

1. ROI 绘制
2. 基于目标检测模型的区域识别

#### 9.4.3 图像处理实例弹窗

表单包含：

1. 实例名称
2. 关联图像获取实例
3. 处理方式
4. 对应配置区域
   - ROI 绘制画布
   - 目标检测模型选择
5. 处理结果预览

### 9.5 步骤三：图像检测

#### 9.5.1 页面结构

1. 检测实例列表
2. 新增检测实例按钮

#### 9.5.2 检测实例弹窗

表单包含：

1. 检测实例名称
2. 图像处理实例选择区
   - 支持多选
3. 使用模型选择
   - 点击后打开模型选择二级抽屉

#### 9.5.3 交互规则

1. 一个检测实例可以选择多个图像处理实例。
2. 未选择图像处理实例时不可保存。
3. 未选择模型时不可保存。

### 9.6 运行态

#### 9.6.1 页面目标

面向一线操作工展示当前检测执行结果。

#### 9.6.2 页面结构

1. 顶部工具信息区
2. 当前执行状态区
3. 主结果区
4. 子结果列表区
5. 对应图像展示区

#### 9.6.3 展示原则

1. 主结果优先展示。
2. 子结果支持切换查看。
3. 原始画面或过程画面作为辅助信息展示。

## 10. 检测记录交互设计

### 10.1 检测记录列表页

#### 10.1.1 页面结构

1. 筛选区
   - 时间范围
   - 工具名称
   - 检测实例名称
   - 业务判定结果
2. 列表区
   - 记录编号
   - 工具名称
   - 检测实例名称
   - 触发时间
   - 执行状态
   - 业务判定结果
   - 查看详情

### 10.2 检测记录详情页

#### 10.2.1 页面结构

1. 总记录信息区
2. 业务判定结果区
3. 子结果列表区
4. 图像查看区

#### 10.2.2 子结果展示

每条子结果展示：

1. 子结果名称
2. 来源图像或区域
3. 使用模型
4. 算法输出结果
5. 结果图像

### 10.3 导出交互

#### 10.3.1 导出类型

检测记录导出分为两类：

1. 记录导出
2. 图像导出

#### 10.3.2 记录导出

1. 列表页支持按当前筛选条件导出记录。
2. 详情页支持导出当前记录详情。
3. 导出内容包含总记录信息和子结果信息。
4. 导出点击后进入导出中状态。
5. 导出完成后直接触发浏览器下载。
6. 导出失败时提示失败原因。

#### 10.3.3 图像导出

图像导出的对象为子结果对应图像。

图像导出支持以下场景：

1. 从记录详情页导出当前记录下的图像
2. 从子结果列表按选中子结果导出图像
3. 从列表页按筛选结果批量导出图像
4. 按人工标记为可疑的子结果批量导出图像

#### 10.3.4 子结果人工标记

1. 子结果行提供“标记可疑”操作。
2. 已标记子结果支持取消标记。
3. 子结果列表支持按是否已标记可疑进行筛选。
4. 列表页支持按是否存在可疑子结果进行筛选。

#### 10.3.5 图像导出交互

1. 记录详情页提供“导出图像”入口。
2. 子结果行提供“导出图像”入口。
3. 列表页提供“批量导出图像”入口。
4. 导出时需让用户选择导出范围。
   - 当前记录全部子结果图像
   - 当前选中子结果图像
   - 当前已标记可疑的子结果图像
   - 当前筛选结果下的全部子结果图像
5. 导出时需让用户选择导出内容。
   - 仅图像
   - 图像加子结果信息
   - 图像加人工标记结果
6. 导出点击后进入导出中状态。
7. 导出完成后直接触发浏览器下载。
8. 导出失败时提示失败原因。
